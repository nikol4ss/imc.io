## IMC.io

BMI (Body Mass Index) calculator developed as my first JavaScript project, focusing on basic logic, input handling, and DOM manipulation.

### About

This project is a practical introduction to programming with JavaScript. It allows calculating BMI from weight and height, demonstrating basic data handling, simple calculations, and updating elements on the page.

---

### Installation

1. Clone the repository:

```bash
git clone https://github.com/user/imc.io.git
```

2. Open the index.html file in your browser.
3. Enter your weight and height in the fields and click "Calculate" to see your BMI result.
